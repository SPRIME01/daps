DAPS The DocBook Authoring and Publishing Suite

DocBook Publishing Made Easy with a Single Command

The DocBook Authoring and Publishing Suite (DAPS) lets you create, build and publish documents written in DocBook XML. It contains a set of stylesheets, scripts and makefiles that enable you to create HTML, PDF, EPUB and other formats from DocBook XML with a single command. It also contains tools to generate profiled source tarballs for distributing your XML sources for translation or review. DAPS also includes tools that assist you when writing DocBook XML, such as linkchecker, validator, spellchecker, editor macros and stylesheets for converting DocBook XML.

With the DAPS docmanager you can manage larger documentation projects with multiple authors (maintainers), deadlines and priorities hosted on an SVN server. Docmanager allows you to view and set doc-related properties to files in SVN.
