What You Need

Before you start writing, you need to install a few packages. To ease writing use an editor with XML support. Recommended is Emacs and DAPS version 2.0 or higher. Do the following to install Emacs and DAPS:

  1. Open your terminal and log in as root by entering 

    tux:~> su -

    Note that tux:~> or (root # below) stands for your prompt. Do not type that part.

  2. Enter your root password.

  3. Install Emacs with zypper: 

    root # zypper in emacs

  4. Install DAPS with zypper: 

    root # zypper in daps
